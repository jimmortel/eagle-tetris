# 🦅 Eagle Tetris

A fast-paced, "Eagle-themed" classic Tetris experience built for the Farcaster ecosystem. Part of the **Eagle Arcade** suite on Solana. 🦅🧱🎮

---

## 🚀 Features
- **Classic Gameplay:** Smooth rotations and line clearing.
- **Eagle Aesthetics:** Dark mode UI with Gold & Black accents.
- **Lightweight:** Pure HTML5/JavaScript, no heavy dependencies.
- **Farcaster Ready:** Built to be easily integrated into Frames.

## 🕹️ How to Play
- **Left/Right Arrows:** Move piece
- **Up Arrow / W:** Rotate
- **Down Arrow:** Soft drop
- **Q:** Counter-rotate
